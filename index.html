🚀 セットアップ手順（3ステップ）

Step 1: Pythonライブラリをインストール
pip install yfinance pandas numpy scikit-learn

Step 2: 予測スクリプトを実行
python gold_predictor.py
→ predictions.json が生成される

Step 3: HTMLファイルとJSONを同じフォルダに配置
index.html と predictions.json を同じ場所に置いてブラウザで開く

⏰ 毎日自動更新したい場合:
crontab -e で以下を追加（平日10時に実行）:
0 10 * * 1-5 cd /path && python gold_predictor.py

📁 ファイル構成
📂 gold-predictor/
├── 📄 index.html         ← このファイル
├── 🐍 gold_predictor.py   ← Python予測エンジン
├── 📊 predictions.json   ← 自動生成される
└── 📄 model_report.txt   ← 自動生成される
🌐 デプロイ方法

ローカル: ブラウザで直接開く（JSONは同じフォルダに）

GitHub Pages: リポジトリにpush → Settings → Pages → Deploy

Netlify: ファイルをドラッグ&ドロップで即デプロイ

自動化: GitHub Actions + cron で毎日JSONを更新
