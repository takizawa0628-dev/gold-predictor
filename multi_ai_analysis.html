マルチAI分析ツール

5つのAIが議題を分析・討論し、各自の提案を提出

議題と背景を入力
解決したい課題や検討したいテーマを具体的に入力してください
関連する状況、制約、目標などの背景情報を入力してください
📊 保存済みの分析結果を表示しています。新しい分析を実行するにはPythonスクリプトを使用してください。
💡 使い方
• Pythonスクリプト: python multi_ai_analyzer.py --topic "議題" --background "背景"
• ブラウザ: APIキーを環境変数に設定してこのページから直接分析（対応準備中）
• 結果は analysis_result.json に保存され、このページで自動的に表示されます

⚙️ セットアップ手順

Step 1: 使用するAIのAPIキーを環境変数に設定
export OPENAI_API_KEY="sk-..."   (ChatGPT)
export ANTHROPIC_API_KEY="sk-ant-..."   (Claude)
export GOOGLE_API_KEY="AI..."   (Gemini)
export DEEPSEEK_API_KEY="sk-..."   (DeepSeek)
export XAI_API_KEY="xai-..."   (Grok)

Step 2: Pythonスクリプトを実行
python multi_ai_analyzer.py -t "議題" -b "背景情報"

Step 3: このHTMLで結果を表示
analysis_result.json が同じフォルダにあれば自動読み込み

オプション: 特定のAIだけ使用
python multi_ai_analyzer.py -t "議題" -b "背景" --ais chatgpt claude gemini
